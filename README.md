## ✨ Features

- 🖊️ **Pluma libre** — traza paths a mano alzada
- **Formas**: línea, rectángulo, círculo, elipse, flecha y texto
- 🎨 **Estilos** — color de trazo/relleno, grosor, opacidad y dash (sólido · guiones · puntos)
- ⚡ **Preview en tiempo real** — el SVG se genera mientras dibujas
- 📋 **Copia el código** con un clic
- 💾 **Descarga** el archivo `.svg` listo para usar
- ⌨️ Atajos de teclado: `P` `L` `R` `C` `E` `A` `T` · `Ctrl+Z` para deshacer

## 🚀 Uso

Abre `index.html` en tu navegador — no requiere instalación ni dependencias. Para mayor comodidad puedes instalar la extencion Live Server en VSCode.

```
git clone https://github.com/NoahXD-dev/svg-forge.git
cd svg-forge
# Abre index.html en tu navegador
```

## 🛠️ Herramientas

| Tecla | Herramienta |
|-------|-------------|
| `P` | Pluma libre |
| `L` | Línea |
| `R` | Rectángulo |
| `C` | Círculo |
| `E` | Elipse |
| `A` | Flecha |
| `T` | Texto |
| `Ctrl+Z` | Deshacer |
